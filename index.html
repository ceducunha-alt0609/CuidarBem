📲 Instalar CuidarBem
🤖 Android / Chrome no PC
1. Abra o arquivo em um servidor local ou pelo Chrome
2. Clique nos 3 pontinhos (⋮) no canto superior direito
3. Toque em "Instalar app" ou "Adicionar à tela inicial"
4. Confirme a instalação
🍎 iPhone / iPad (Safari)
1. Abra este arquivo no Safari
2. Toque no ícone de compartilhar ⬆️ na barra inferior
3. Role para baixo e toque em "Adicionar à Tela de Início" ＋
4. Confirme tocando em "Adicionar"
💡 Dica: Para instalar como app, o arquivo precisa estar hospedado num servidor (ex: aberto pelo VS Code Live Server, XAMPP, ou enviado para o celular via GitHub Pages / Netlify Drop). No iPhone, basta o Safari.
